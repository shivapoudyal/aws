----------------------------------- S3 BUCKETS TYPES --------------------------------

1) S3 (Simple Store Service) OBJECT BUCKET -> can store any types of data ie. -> music, text, sql, video etc...
2) EFS (ELASTIC FILE SYSTEM)
3) EBS (ELASTIC BLOCK STORAGE) -> it connected with ec2 machine.
4) S3 GLACIER -> it used to store big unused (not frequently used) data, it cost is too low.
5) SNOWBALL -> Snowball is a petabyte-scale data transport solution that uses secure appliances to transfer large amounts of data into and out of the AWS cloud
